# CLI Utilities

A collection of command-line tools written in Go.

- Get-IPv4
- Get-IPInfo

## Installation

```sh
go install github.com/chtozamm/cli-utilities/cmd/get-ipv4
go install github.com/chtozamm/cli-utilities/cmd/get-ipinfo
```

## Usage

```sh
Get-IPv4
```

```sh
Get-IPInfo
```
